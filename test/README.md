# 测试文件夹

这个文件夹包含用于测试食堂管理系统的各种脚本和工具。

## 文件说明

- `add_sample_dishes.py` - 为数据库添加示例菜品数据
- `test_ai_recommendation.py` - 测试AI推荐功能
- `test_api.py` - 测试API接口

## 使用方法

1. 添加示例菜品数据：
```bash
cd canteen_new
python ../test/add_sample_dishes.py
```

2. 测试AI推荐功能：
```bash
cd canteen_new
python ../test/test_ai_recommendation.py
```

3. 测试API接口：
```bash
cd canteen_new
python ../test/test_api.py
